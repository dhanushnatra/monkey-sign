## **Installation and Usage**

**Linux**

```bash
python3.12 -m venv env
source env/bin/activate
python3 -m pip install -r requirements.txt
python3 main.py
```

**Windows**

```bat
python3.12 -m venv env
cd env/Scripts && activate && cd ../../
python3 -m pip install -r requirements.txt
python3 main.py

```
